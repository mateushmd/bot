# Simulador de Labirinto

+ Disponível em: https://mateushmd.github.io/bot/

+ Nesse simulador, o usuário pode testar 6 algoritmos de resolução de labirinto diferentes, em um labirinto customizável. Os métodos disponíveis são:
    1. A* (Heurística distância de Manhattan)
    2. A* (Heurística distância Euclideana)
    3. A* (Heurística distância de Chebyshev)
    4. A* (Heurística aleatória)
    5. Dijkstra
    6. Busca em largura
+ Para editar o labirinto, basta clicar com o botao esquerdo do mouse para criar paredes e com o botão direito para destruí-las. A posição inicial do labirinto, indicada pelo quadrado vermelho, também pode ser editada com as caixas de entrada de texto da interface.
